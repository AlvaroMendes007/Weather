## Sobre o Projeto

Projeto de Previsão do tempo feito em React usada para trazer a temperatura da cidade inserida buscando em uma API.

### Rodar local

- clonar repositório
- npm install
- digitar a url localhost:3000 e ver a mágica acontecer

### Link para acesso

https://weather-woad.vercel.app/
